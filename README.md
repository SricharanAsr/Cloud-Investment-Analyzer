# InvestVision AI — Cloud-Native Portfolio Analyzer

InvestVision AI is a sophisticated, 100% cloud-native application designed to digitize, analyze, and store investment portfolio data from screenshots. Using **Azure AI Vision** for high-precision OCR and a serverless **Azure Functions** backend, it transforms unstructured images into actionable financial intelligence.

## 🚀 Key Features
- **AI-Powered OCR**: Automatically extracts stock symbols, quantities, and prices from portfolio screenshots.
- **Financial Intelligence**: Calculates/assigns **P/E and P/B ratios** for identified holdings.
- **Smart Suggestions**: Provides AI-driven investment advice (**Buy, Hold, or Avoid**) based on valuation metrics.
- **Premium Dashboard**: A sleek, dark-mode web interface with glassmorphism design, hosted on **Azure Storage**.
- **Serverless Architecture**: Built entirely on Azure (Functions, Cosmos DB, AI Vision, and Blob Storage).

## 🛠️ Technology Stack
- **Frontend**: Vanilla JS, HTML5, CSS3 (Glassmorphism UI).
- **Backend**: Python 3.11 with Azure Functions.
- **Database**: Azure Cosmos DB (NoSQL).
- **AI Service**: Azure AI Vision (Optical Character Recognition).
- **Hosting**: Azure Blob Storage (Static Website Hosting).
- **Security**: Azure Key Vault for secret management.

## 📦 Project Structure
- `function_app.py`: Core backend logic for OCR processing and financial analysis.
- `index.html`: Premium cloud dashboard.
- `setup_azure.ps1`: Automated infrastructure deployment script.
- `requirements.txt`: Python dependencies for the backend.
- `host.json`: Azure Functions configuration.

## 📂 Getting Started
1. **Cloud Setup**: Run `setup_azure.ps1` to provision all necessary Azure resources.
2. **Backend**: Deploy the Python logic using Azure Functions Core Tools:
   ```bash
   func azure functionapp publish <your-function-name>
   ```
3. **Frontend**: Access the dashboard via the static website URL generated by the Azure Storage account.
