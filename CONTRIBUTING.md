# Contributing to InvestVision AI

Thank you for your interest in contributing to InvestVision AI! We welcome contributions to improve the AI precision, UI/UX, or overall cloud architecture.

## How to Contribute

1.  **Fork the Repository**: Create a personal fork of the project on GitHub.
2.  **Clone the Fork**: Clone your fork to your local machine.
3.  **Create a Branch**: Create a new branch for your feature or bug fix.
4.  **Make Changes**: Implement your changes and ensure they follow the existing coding style.
5.  **Test**: Verify your changes locally.
6.  **Commit**: Commit your changes with clear and descriptive messages.
7.  **Push**: Push your branch to your GitHub fork.
8.  **Pull Request**: Open a pull request against the `main` branch of the original repository.

## Coding Guidelines
- Use descriptive names for variables and functions.
- Add comments and docstrings where necessary.
- Ensure PowerShell scripts are documented.
- Maintain the glassmorphism aesthetic for frontend changes.

## Reporting Issues
If you find a bug or have a feature request, please open an issue on the GitHub repository.
